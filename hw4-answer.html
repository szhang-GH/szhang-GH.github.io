  1. In homework # 3 you have created the projection matrix \mathbf{H} using the cars dataset.

    1. Compute SSE, SSR, and SST using the response vector \mathbf{Y}, and the following matrices \mathbf{I}, \mathbf{H}, \mathbf{J}=\mathbf{1}\mathbf{1}'.

      Answer:

      # Load data
data(cars)
n <- nrow(cars)
Y <- cars$dist

# Construct Design Matrix X and Hat Matrix H from HW3
X <- cbind(1, cars$speed)
inv_XX <- solve(t(X) %*% X)
H <- X %*% inv_XX %*% t(X)

# Define I and J
I <- diag(n)
J <- matrix(1, n, n)

# Compute SST = Y'(I - 1/n J)Y
# J = 11' so 1/n J is the centering matrix component
SST <- t(Y) %*% (I - (1/n) * J) %*% Y

# Compute SSE = Y'(I - H)Y
SSE <- t(Y) %*% (I - H) %*% Y

# Compute SSR = Y'(H - 1/n J)Y (Regression Sum of Squares)
SSR <- t(Y) %*% (H - (1/n) * J) %*% Y

cat("SST =", SST, "\n")
      SST = 32538.98 
      cat("SSR =", SSR, "\n")
      SSR = 21185.46 
      cat("SSE =", SSE, "\n")
      SSE = 11353.52

    3. What are the distribution of SSE, SSR, and SST?

      Answer:

      Under the standard linear regression assumptions (errors are normally distributed):

      • SSE: The error sum of squares scaled by the error variance follows a chi-square distribution with n-p degrees of freedom: \frac{SSE}{\sigma^2} \sim \chi^2_{n-p} where n=50 and p=2, so df_E = 48.

      • SSR: Under the null hypothesis H_0: \beta_1 = 0, the regression sum of squares scaled by the error variance follows a chi-square distribution with p-1 degrees of freedom: \frac{SSR}{\sigma^2} \sim \chi^2_{p-1} where p-1 = 1.

      • SST: Under the null hypothesis H_0: \beta_1 = 0, the total sum of squares scaled by the error variance follows a chi-square distribution with n-1 degrees of freedom: \frac{SST}{\sigma^2} \sim \chi^2_{n-1} where n-1 = 49.

    4. Estimate Var(\mathbf{b}), the variance-covariance matrix of the estimated regression coefficients \mathbf{b}. Estimate the correlation between b_0 and b_1.

      Answer:

      # Degrees of freedom
p <- 2
df_E <- n - p

# Estimate error variance (MSE) = s^2
s2 <- as.numeric(SSE / df_E)

# Estimated Variance-Covariance Matrix: Var(b) = s^2 * (X'X)^-1
Var_b <- s2 * inv_XX
print(Var_b)
                [,1]       [,2]
[1,] 45.676514 -2.6588234
[2,] -2.658823  0.1726509
      # Correlation between b0 (Intercept) and b1 (speed)
# cov(b0, b1) is the off-diagonal element
# var(b0) and var(b1) are diagonal elements
cov_b0_b1 <- Var_b[1, 2]
var_b0 <- Var_b[1, 1]
var_b1 <- Var_b[2, 2]

cor_b0_b1 <- cov_b0_b1 / sqrt(var_b0 * var_b1)

cat("Estimated Correlation between b0 and b1:", cor_b0_b1, "\n")
      Estimated Correlation between b0 and b1: -0.9468008

    5. Conduct an F test to evaluate the significance of b_1. Report the null and alternative hypotheses, the test statistic, the degrees of freedom, the p-value, and your conclusion.

      Answer:

      Hypotheses: H_0: \beta_1 = 0 \quad \text{vs.} \quad H_1: \beta_1 \neq 0

      Test Statistic and P-value:

      # Degrees of freedom for Regression
df_R <- p - 1

# Mean Squares
MSR <- as.numeric(SSR / df_R)
MSE <- s2

# F Statistic
F_stat <- MSR / MSE

# P-value from F distribution
p_value_F <- 1 - pf(F_stat, df_R, df_E)

cat("F Statistic:", F_stat, "\n")
      F Statistic: 89.56711 
      cat("Degrees of Freedom:", df_R, "(numerator) and", df_E, "(denominator)\n")
      Degrees of Freedom: 1 (numerator) and 48 (denominator)
      cat("P-value:", p_value_F, "\n")
      P-value: 1.489919e-12

      Conclusion: The F statistic is 89.57 with 1 and 48 degrees of freedom, yielding a p-value of 1.49 \times 10^{-12}. Since the p-value is effectively zero (and < 0.05), we reject the null hypothesis H_0. We conclude that there is a statistically significant linear relationship between speed and stopping distance.
